# STROPSE ⚡
**India's Unified Esports Ecosystem**

A premium, high-performance platform designed for competitive gamers to play, improve, and dominate. Built with React.js + Vite and powered by Firebase.

## Core Features
1. **Tournament Hub**: Browse and register for live, past, and upcoming tournaments.
2. **Player Card Engine**: Mint dynamic, aesthetic-driven game profiles based on K/D, Win Rates, or Elo scores.
3. **Community Hub**: Interconnected chat rooms for players to find teams and hang out.
4. **Admin Dashboard**: Restricted control center to orchestrate the entire ecosystem.

## Tech Stack
- Frontend: React + Vite (Vanilla CSS)
- Backend: Firebase (Auth, Firestore)
- Design: High-contrast Cyberpunk / Neo-Brutalism
